CSV Extractor

This is a very simple program that I made for my dad. I realized that several times every year he would spend several hours searching through a CSV file and copying all the emails of people he wanted to send holiday cards or open house invites to. I thought for a minute and then said, "I can make a Python program to do that for you." And so I did. It searches the file and if "Holiday Card" or "Open House" is found in the row, it adds their email to a text document. This program runs in about 30 seconds and from there my dad can just copy the emails and send the email. Instead of taking several hours, it takes a few minutes now. View the full source code on GitHub at this link: https://github.com/HateAndLoveModding/CSV_Extractor.
